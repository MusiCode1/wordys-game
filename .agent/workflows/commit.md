---
description: תהליך ביצוע קומיט מסודר לגיט
---
1. בדוק את הסטטוס הנוכחי באמצעות הרצת הפקודה `git status`.
2. הוסף ל-Stage **רק** את הקבצים הרלוונטיים ששונו או נוצרו במסגרת המשימה הנוכחית באמצעות `git add <files>`. **אין להשתמש ב-`git add .` באופן אוטומטי.**
3. נסח הודעת קומיט ברורה ותמציתית בעברית, המתארת את השינויים שבוצעו.
4. בקש את אישור המשתמש להודעת הקומיט ולרשימת הקבצים שבחרת להוסיף.
5. לאחר האישור, בצע את הקומיט באמצעות `git commit -m "ההודעה שאושרה"`.
